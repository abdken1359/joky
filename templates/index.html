All Jokes

To get all jokes, call the following endpoint without any query parameters.

/api/jokes

The response body will look like this:

{
    jokes:[
        {   
            category:string,
            headline:string,
            text:string,
        }
    ]
}

Joke id

Endpoint:

/api/jokes/{id:int}

Response body:

{
   joke:{
    category:string,
    headline:string,
    text:string,
   }
}
If the id don't point to a joke, a 404 response code will be issued. Also, if the id isn't a positive integer, a 404 response code will be issued.

Joke Categories

Endpoint:

/api/jokes/categories

Response body:

{
   categories:string[]
}
