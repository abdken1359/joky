All Jokes

Get all jokes

Endpoint:

/api/jokes

Response body: 

{
    jokes:[
        {   
            category:string,
            headline:string,
            text:string,
        }
    ]
}

Joke id

Get a joke from a specified id

Endpoint:

/api/jokes/{id:int}

Response body:

{
   joke:{
    category:string,
    headline:string,
    text:string,
   }
}
If the id don't point to a joke, a 404 response code will be issued. Also, if the id isn't a positive integer, a 404 response code will be issued.

Joke Limit

Get jokes with a specified limit

Endpoint:

/api/jokes/?limit="1"

Response body:

{  
   limit:query.limit
   jokes:[
    {
    category:string,
    headline:string,
    text:string,
   }
   ]
}
